# AutoDarkMin 用户指南

## 时间选择

### 浅色开始时间
- **功能**: 设置浅色模式的开始时间。
- **操作**: 在指定的时间点后，系统将自动切换到浅色模式。

### 深色开始时间
- **功能**: 设置深色模式的开始时间。
- **操作**: 在指定的时间点后，系统将自动切换到深色模式。

## 设置

### 开机自启动
- **功能**: 设置程序在系统启动时自动运行。
- **操作**: 勾选此选项以启用开机自启动功能。

### 强制浅色
- **功能**: 强制系统始终使用浅色模式。
- **操作**: 选择此选项以强制系统保持浅色模式。

### 强制深色
- **功能**: 强制系统始终使用深色模式。
- **操作**: 选择此选项以强制系统保持深色模式。

### 启用窗口置顶
- **功能**: 启用窗口置顶功能。
- **操作**: 
  - 勾选此选项以启用窗口置顶功能。
  - 双击状态栏图标可以开启置顶窗口捕获。
  - 在窗口捕获状态下，可以通过按下 **ESC 键** 或点击 **鼠标右键** 退出捕获模式。
  - 右键点击状态栏图标可显示更多操作菜单。
